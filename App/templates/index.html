📘 About This Project

This machine learning system intelligently predicts whether a health insurance claim is likely to be approved or rejected based on applicant and claim details.

📊 Input Features

Data-driven evaluation using age, BMI, claim type, doctor visits, etc.

🧠 ML Algorithms

Powered by models like Logistic Regression, SVM, and Gradient Boosting.

🎯 Decision Output

Predicts whether a submitted claim is likely to be approved or rejected.

📖 Understanding the Fields

  • Age: Policyholder's age.
  • Gender: 1 = Male, 0 = Female.
  • BMI: Body Mass Index of the applicant.
  • Smoker: 1 = Yes, 0 = No.
  • Number of Dependents: Family members dependent on the policy.
  • Type of Claim: Encoded category of the claim.
  • Claim Amount: Monetary amount requested for approval.
  • Previous Claims: Number of claims filed previously.
  • Stay Duration: Days spent hospitalized.
  • Doctor Visits: Total consultations made for this claim.
